# my-vue-app
透過 Github Action，在 vue 專案推到 main 後自動編譯及更新 S3 上的網站。

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve

### Compiles and minifies for production

npm run build
```

### Lints and fixes files

npm run lint
